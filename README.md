# Jetton contracts
Contracts copied from [ton-blockchain/token-contract](https://github.com/ton-blockchain/token-contract/tree/main/ft)

## Installation
```shell
yarn install && yarn mnemonic
```

## Compilation
```shell
yarn compile
```

## Generate new mnemonic in `.env`
```shell
yarn mnemonic
```

## View wallet address and balance
```shell
yarn wallet
```

## Deploy
```shell
yarn deploy
```

## Mint
```shell
yarn mint
```
